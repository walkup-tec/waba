# LOG — Card Restrição Temporária com cores de Desconectados

## Contexto

O card de resumo **Restrição Temporária** usava amarelo (igual a Preparando). O pedido foi alinhar as cores às de **Instâncias desconectadas**.

## Solução

- Borda esquerda: `rgba(248, 113, 113, 0.55)` (mesma de Desconectados)
- Valor numérico: `#fca5a5`
- Preparando permanece amarelo `#fbbf24`

- `index.html`
- `dist/index.html`
