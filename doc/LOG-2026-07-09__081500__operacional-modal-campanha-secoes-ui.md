# UI — modal detalhes campanha operacional por seções

**Data:** 2026-07-09

## Pedido

- Organizar modal em seções (prints do usuário).
- Subtítulo só com **nome da campanha** (remover `· assinante UUID`).
- **Baixar imagem** e **Baixar leads** junto com opções de texto e link (materiais).
- Botões de **ação** (BM inoperante, Reportar erro) separados dos downloads.

## Layout

1. **Assinante** — ID + segmento (grid 2 colunas)
2. **Campanha** — nome, DDD, envios, status
3. **Registro** — criada em, atualizada em
4. **Materiais da campanha** — textos, link, downloads
5. **Ações** — BM inoperante, Reportar erro
6. **Rodapé** — Confirmar início / erro / Fechar
